La solution

Domolab est un projet basé sur le projet MySensors, celui ci permet d'avoir une solution domotique DIY. Le point fort de ce projet, c'est qu'il est basé sur Arduino et communique sans fils avec les capteurs.

  • Open Source MySensors est un projet Open Source qui permet à chacun de contribuer et d'avoir un projet évolutif et perein dans le temps.

  • Peu onéreux Le projet se veut peu onéreux et indépendant en comparaison à des solutions propritaires (Fibaro, Zwave, Enoncean, etc ...)

  • Arduino MySensors utilise les micro-controleurs pour sa simplicité, faible consomation (suivant le modèle) et son faible coût

  • Radio Afin de simplifier sont installation, MySensors comminique avec les capteurs par onde Radio. Pour cela, il utilise le composant NRF24L01+ de Nordic semiconductor

  • Interopérable MySensors peut communiquer avec une grande majorité de solutions de domotiques.

  • Evolutif Si un élement n'existe pas dans MySensors. il est facile de le créer avec le couple Arduino + NRF24L01+.
