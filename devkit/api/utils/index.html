Utilities

For plugins there are some useful utilities provided with Lektor. This list will only grow with time.

build_url()

Helper function that assists in building URL paths.

get_structure_hash()

Generates a hash from a Python structure.

join_path()

Joins two Lektor paths together correctly.

parse_path()

Parses a path into components.

process_image()

Build an image from a source image, optionally compressing and resizing.

url_to()

Calculates the URL to a specific page relative to the current.
