    IDE Arduino

    L'IDE Arduino permet de programmer(flasher) les modules arduino qui permetront de fabriquer des capteurs ou gateway.

    Installation

    • Télécharger la version 1.6 de l'IDE Arduino depuis le gestionaire de paquet de votre distribution ou l'installer depuis les sources
    • Vérifier ou sont installé les librairies utilisateurs dans les préférence de l’interface Arduino ( Fichier ➤ Préférences ).
    • Installer le kit de developpement de MySensors
      • Ouvrir un terminal
      • cd ~/Arduino # Voir chemin capture écran précédente
wget https://github.com/mysensors/Arduino/archive/master.zip
bsdtar -xf master.zip -s'|[^/]*/||'
